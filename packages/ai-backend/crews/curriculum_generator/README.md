# Curriculum Generator Crew

This crew is responsible for generating the curriculum for the Professor Pixel platform. It is a migration of the original "rswarm orchestration" from the `professor-pixels-arcade-academy` repository.

## Responsibilities

- Generate new lessons and cartridges based on the curriculum schema.
- Ensure that the generated content is engaging, educational, and consistent with the Professor Pixel character.
- Validate the generated content against the curriculum schema.
